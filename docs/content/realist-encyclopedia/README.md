# Realist.ca Investor Encyclopedia Content Pack

First batch of searchable encyclopedia content for Canadian real estate investors. This is content-only by design: no risky frontend rewrite, no database migration, no deployment.

## Files

- `guides.json` — 74 investor-heavy guide entries.
- `tool-specs.json` — 18 calculator/tool specs.
- `manifest.json` — searchable index, categories, related slugs, and tool map.
- `IMPLEMENTATION_PLAN.md` — how Replit/Codex should wire this into the site.

## Guide schema

Each guide includes: title, slug, summary, definition, why it matters, formula where relevant, example, investor interpretation, common mistakes, related terms, Realist tie-in, source/caveat notes, category, tags, difficulty, search keywords, and optional tool spec slug.

## Language standard

Investor-heavy, practical, searchable. Every term should answer: what changes for someone trying to buy, finance, operate, sell, refinance, or avoid a deal?

## Do not treat as legal/tax/lending advice

The pack is educational. Tax, legal, lender, insurer, provincial, municipal, and market rules need live verification before launch.
