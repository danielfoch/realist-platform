# Realist.ca Investor Encyclopedia Content Pack

Searchable encyclopedia content for Canadian real estate investors. This is content-only by design: no risky frontend rewrite, no database migration, no deployment.

Batch 2 adds 75 investor-heavy entries across transaction law, planning/entitlements, landlord-tenant operations, Canadian tax structures, construction lending, condo diligence, valuation, environmental risk, and commercial leasing.

## Files

- `guides.json` — 149 investor-heavy guide entries.
- `tool-specs.json` — 28 calculator/tool specs.
- `manifest.json` — searchable index, categories, related slugs, and tool map.
- `IMPLEMENTATION_PLAN.md` — how Replit/Codex should wire this into the site.

## Guide schema

Each guide includes: title, slug, summary, definition, why it matters, formula where relevant, example, investor interpretation, common mistakes, related terms, Realist tie-in, source/caveat notes, category, tags, difficulty, search keywords, optional tool spec slug, status, canonical path, and section order.

## Language standard

Investor-heavy, practical, searchable. Every term should answer: what changes for someone trying to buy, finance, operate, sell, refinance, or avoid a deal?

## Do not treat as legal/tax/lending advice

The pack is educational. Tax, legal, lender, insurer, provincial, municipal, and market rules need live verification before launch.
