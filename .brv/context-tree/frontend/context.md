# Domain: frontend

## Purpose
Contains all knowledge related to the Realist Platform frontend, including components, pages, forms, and UI/UX patterns.

## Scope
Included in this domain:
- React components and pages
- Form implementations and validation
- CSS and design systems
- User flows and wizards
- UI state management

Excluded from this domain:
- Backend API routes (belongs in backend)
- Database schemas (belongs in data_models)
