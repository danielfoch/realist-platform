# Domain: growth

## Purpose
Contains knowledge about viral growth mechanics, referral systems, and user acquisition features designed to drive platform adoption.

## Scope
Included in this domain:
- Viral sharing mechanisms and referral links
- Credit/reward systems for user engagement
- Growth analytics and funnel optimization
- User coaching and conversion insights

Excluded from this domain:
- General authentication (belongs in auth domain)
- Database schema details (belongs in architecture domain)

## Usage
Use this domain for documenting viral loops, referral programs, credit systems, and growth-related features.
