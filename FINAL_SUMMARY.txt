API ENDPOINTS CREATED
================================================================================

✅ GET /api/listings
   - Search with filters (city, province, price, bedrooms, type)
   - Pagination (max 100 per page)
   - Investment focus mode
   - Example: /api/listings?city=Toronto&investmentFocus=true

✅ GET /api/listings/:mlsNumber
   - Get single listing with full details
   - Includes photos, agent, brokerage, history
   - Example: /api/listings/C12345678

✅ GET /api/listings/investment/top
   - Top investment properties by cap rate
   - Filter by city/province
   - Example: /api/listings/investment/top?limit=50

✅ GET /api/listings/map
   - Listings for map view (coordinates + basic info)
   - Geospatial bounds filtering
   - Example: /api/listings/map?bounds=43.6,-79.5,43.7,-79.3

✅ GET /api/stats
   - Market statistics (avg price, cap rates, counts)
   - Filter by city/province
   - Example: /api/stats?city=Toronto
