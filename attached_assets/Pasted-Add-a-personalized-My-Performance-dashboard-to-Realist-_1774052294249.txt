Add a personalized “My Performance” dashboard to Realist.ca and connect it to the CTA link: “Find out how you compare”.

GOAL:
When a user clicks the CTA in the email, they land on login (if not authenticated) and are then redirected to a dashboard showing their personal KPIs vs platform averages.

---

ROUTING:

1. Create route:
/my-performance

2. If user NOT logged in:
- Redirect to /login
- After login → redirect back to /my-performance

---

BACKEND:

Create endpoint:

GET /api/user-performance

Return:

{
  user: {
    deals_analyzed: number,
    avg_cap_rate: number,
    avg_cash_on_cash: number,
    avg_irr: number,
    rank_weekly: number,
    rank_all_time: number
  },
  platform: {
    avg_cap_rate: number,
    avg_cash_on_cash: number,
    avg_irr: number,
    total_deals_analyzed: number
  },
  leaderboard: {
    weekly_top_3: [],
    all_time_top_3: []
  }
}

---

DATA CALCULATION:

USER METRICS:
- deals_analyzed = count of deals by user
- avg_cap_rate = avg of user deal analyses
- avg_cash_on_cash = avg of user deal analyses
- avg_irr = avg of user deal analyses

PLATFORM METRICS:
- avg_cap_rate = avg across all users (last 7 days)
- avg_cash_on_cash = avg across all users
- avg_irr = avg across all users

RANKINGS:
- rank_weekly = position based on deals analyzed in last 7 days
- rank_all_time = position based on total deals analyzed

Create clean dashboard with sections:
