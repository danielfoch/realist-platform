4) Financial Logic (Define Clearly in Code)
Effective Gross Income (EGI)

Gross rent = (market_units * market_rent + affordable_units * affordable_rent) * 12

Vacancy loss = gross rent * vacancy_rate

EGI = gross rent - vacancy_loss + other_income

NOI

OpEx = either:

expense_ratio * EGI, or

sum(itemized)

NOI = EGI - OpEx

Debt Service + DSCR

Use standard mortgage payment formula (annualized):

Payment = P * r / (1 - (1+r)^-n)

r = monthly rate

n = amortization_months

Annual Debt Service = monthly_payment * 12

DSCR = NOI / Annual Debt Service

Loan Amount

For acquisition: Loan = LTV * purchase_price

For new build: allow toggle “use LTC instead of LTV”

Loan = LTC * total_project_cost

Total project cost = land + hard + soft + contingency + fees

CMHC Insurance Premium

Use config table keyed by point tier:

premium % applied to base loan amount (or show both “net loan” and “gross loan”)

Net loan amount after premium/fees displayed clearly.

Keep this modular so rates can be updated without code changes.
